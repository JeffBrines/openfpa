# pyfpa 13-Week Cash Engine (cash13) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `pyfpa/cash13/` — a schedule-driven 13-week direct-method cash forecast that turns a list of scheduled weekly receipts and disbursements into a week-by-week ending-cash trajectory, plus a runway summary answering "when do we run out of money."

**Architecture:** Pure functions, no I/O. A `Cash13Config` (pydantic) holds opening cash and lists of `WeeklyFlow` items (with `once`/`weekly`/`biweekly` recurrence). `expand_flow` turns one flow into a per-week amount array; `cash13_forecast` sums receipts/disbursements into a weekly DataFrame with cumulative ending cash; `runway_summary` reports the trough and first negative week. This is the operationally-real shape of a 13-week TWCF tool (scheduled flows), not a smoothed monthly projection.

**Tech Stack:** Python 3.11+, pandas, pydantic v2, pytest. Builds on the existing `pyfpa` package (Plan 1).

**Scope note:** This plan is the cash13 engine only. The IO layer (CSV/XLSX in, Excel/briefing out) and data-source adapters (NetSuite/QuickBooks/Shopify) are Plan 3. A monthly↔weekly reconciliation bridge is also deferred to Plan 3 (it depends on the IO/derivation layer).

```python
from __future__ import annotations

from typing import Literal

from pydantic import BaseModel, Field, model_validator


class WeeklyFlow(BaseModel):
    """A scheduled cash flow. `amount` is a magnitude (>=0); whether it is a
    receipt or disbursement is determined by which list it lives in."""

    name: str
    amount: float = Field(ge=0)
    start_week: int = Field(ge=1)
    recurrence: Literal["once", "weekly", "biweekly"] = "once"
    end_week: int | None = Field(default=None, ge=1)

    @model_validator(mode="after")
    def _end_after_start(self) -> "WeeklyFlow":
        if self.end_week is not None and self.end_week < self.start_week:
            raise ValueError("end_week must be >= start_week")
        return self


class Cash13Config(BaseModel):
    opening_cash: float
    weeks: int = Field(default=13, ge=1, le=52)
    receipts: list[WeeklyFlow] = Field(default_factory=list)
    disbursements: list[WeeklyFlow] = Field(default_factory=list)
```

```python
from __future__ import annotations

from pyfpa.cash13.schemas import WeeklyFlow


def expand_flow(flow: WeeklyFlow, weeks: int) -> list[float]:
    """Expand one WeeklyFlow into a length-`weeks` list (index 0 == week 1)."""
    amounts = [0.0] * weeks
    end = flow.end_week if flow.end_week is not None else weeks

    if flow.recurrence == "once":
        hit_weeks = [flow.start_week]
    elif flow.recurrence == "weekly":
        hit_weeks = range(flow.start_week, end + 1)
    else:  # biweekly
        hit_weeks = range(flow.start_week, end + 1, 2)

    for week in hit_weeks:
        if 1 <= week <= weeks:
            amounts[week - 1] += flow.amount
    return amounts
```

```python
from __future__ import annotations

import pandas as pd

from pyfpa.cash13.flows import expand_flow
from pyfpa.cash13.schemas import Cash13Config, WeeklyFlow


def _sum_flows(flows: list[WeeklyFlow], weeks: int) -> list[float]:
    totals = [0.0] * weeks
    for flow in flows:
        for i, amount in enumerate(expand_flow(flow, weeks)):
            totals[i] += amount
    return totals


def cash13_forecast(cfg: Cash13Config) -> pd.DataFrame:
    """Weekly direct-method cash forecast: receipts, disbursements, net, and
    cumulative ending cash. Raw cash position — no auto-draws."""
    weeks = cfg.weeks
    receipts = _sum_flows(cfg.receipts, weeks)
    disbursements = _sum_flows(cfg.disbursements, weeks)
    net_cash = [r - d for r, d in zip(receipts, disbursements)]

    ending_cash = []
    balance = cfg.opening_cash
    for n in net_cash:
        balance += n
        ending_cash.append(balance)

    idx = pd.RangeIndex(start=1, stop=weeks + 1, name="week")
    return pd.DataFrame(
        {
            "receipts": receipts,
            "disbursements": disbursements,
            "net_cash": net_cash,
            "ending_cash": ending_cash,
        },
        index=idx,
    )
```

```python
from __future__ import annotations

import pandas as pd


def runway_summary(forecast_df: pd.DataFrame) -> dict:
    """Summarize a 13-week forecast: cash trough and first negative week.

    Returns {"min_cash": float, "min_week": int, "first_negative_week": int | None}.
    """
    ending = forecast_df["ending_cash"]
    negative_weeks = ending.index[ending < 0]
    first_negative = int(negative_weeks[0]) if len(negative_weeks) > 0 else None
    return {
        "min_cash": float(ending.min()),
        "min_week": int(ending.idxmin()),
        "first_negative_week": first_negative,
    }
```
